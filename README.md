# 📧 Форма обратной связи на Java Spring Boot

## Функционал
- HTML-форма (имя, email, сообщение)
- Валидация на клиенте (JavaScript)
- Отправка данных через fetch()
- Сервер на Spring Boot принимает и возвращает JSON
- Обработка ошибок 400 и 500
- Вывод сообщения об успешной отправке

## Технологии
- **Backend**: Java 17, Spring Boot 3.2, Maven
- **Frontend**: HTML5, CSS3, JavaScript (Fetch API)
- **Валидация**: Jakarta Validation + кастомная клиентская
